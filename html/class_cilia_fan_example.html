void CiliaFanExample.OnTriggerEnter ( Collider  aCollision)
private

Turns on a smell. When an object with the right MainCharacterName colides with an object that has this class and a trigger collider, turns on all fans on the Cilia in the currect SurroundPosition group with the right Smell. Intensity is set to FanSpeed.
