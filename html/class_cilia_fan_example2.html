void CiliaFanExample2.OnTriggerStay ( Collider  aCollision)
private

Turns on a smell to an intensity based on how close the player is to the center of the collider. While an object with the right MainCharacterName colides with an object that has this class and a trigger collider, turns on all fans on the Cilia in the currect SurroundPosition group with the right Smell. Intensity is determined by how close the character is to the middle of the collider.

Parameters
objectthat triggered a collision with us.
